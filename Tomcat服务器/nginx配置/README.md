### or

```php
[root@lb01 nginx-1.6.2]# cat -n conf/nginx.conf
worker_processes  1;

events {
    worker_connections  1024;
}

http {
    include       mime.types;
    default_type  application/octet-stream;
    sendfile        on;
    upstream lbproxy {
        server 10.0.0.12:80 weight=5;
        server 10.0.0.13:80 weight=5;
        server 10.0.0.11:80 backup;
    }
    keepalive_timeout  65;
    server {
        listen       80;
        server_name  lb.liang.com;
        location / {
            root   html;
            index  index.html index.htm;
            proxy_pass http://lbproxy;
        }
    }
}
```

## nginx gzip 小优化：

```php
server {
    gzip on;
    gzip_buffers 32 4k;
    gzip_comp_level 6;
    gzip_min_length 200;
    gzip_types text/css text/xml application/javascript;
    gzip_vary on;
}
```

ngonx 开启 gzip，加快首屏渲染时间，图片并不适用
